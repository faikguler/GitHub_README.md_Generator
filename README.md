# GitHub README.md Generator

A Python tool that generates professional README.md files for GitHub projects through an interactive command-line interface.

## Features

- ✅Interactive command-line questions for project details
- ✅Input validation (prevents empty fields)
- ✅License selection from a list of common open source licenses
- ✅Automatic README.md file generation
- ✅Clean and organized project structure
- ✅No additional packages required. This tool uses only Python standard library.
